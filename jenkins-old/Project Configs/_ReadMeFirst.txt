These are snapshots of the Jenkins project configurations.
They can be used to restore the Projects within a Jenkins instance or as reference.
Since these are snapshots, they are not garenteed to be up to date.  However they will provide a good starting point.

Start with a jenkins-persistent instent-app in OpenShift

Add these Jenkins plug-ins:
- Environment Injector Plugin
- GitHub plugin
- Job Configuration History Plugin
- Promoted Builds Plugin

Install all plug-in updates.
